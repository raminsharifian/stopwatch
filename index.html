Stopwatch Guide

Welcome to the Stopwatch App!

This application allows you to control a stopwatch with the following features:

  • Start/Pause: Click the stopwatch or press "Space" or "P" to toggle between start and pause.
  • Reset: Press the "R" key to reset the stopwatch.
  • Automatic State Saving: Your stopwatch's state is saved automatically in the browser's localStorage, so it persists even when you refresh the page.

Follow the instructions to use the stopwatch efficiently. Happy timing! Developed with ♥ by raminsharifian on GitHub
